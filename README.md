# NikeMania

이 앱은 비공식 앱으로서 

나이키매니아(www.nikemania.com) 사이트를

편리하게 이용하기 위한 브라우저입니다.

본 어플은 나이키매니아 광고 정책을 존중합니다.

나이키매니아에서 보여지는 모든 광고를 동일 하게 노출하고

메뉴에 modify 버튼까지도 메뉴에 추가 해 놓았습니다.



주요 기능
1. 모바일 화면 최적화
	- 모바일에서 나이키 매니아 접속시 항상 확대해서 봐야 하는 부분을 개선
2. 이미지 업로드 기능
	- 이미지 업로드 버튼을 상단에 두어 해당 버튼을 클릭하여 이미지를 올리고 나이키매니아에 이미지 방식 [#이미지URL]으로 복사할 수 있습니다.
